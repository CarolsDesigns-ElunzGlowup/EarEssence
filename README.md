# EarEssence 

A simple and clean landing page for showcasing earrings.

🔗 Live site:  
https://carolsdesigns-elunzglowup.github.io/EarEssence/

---

## About the Project
EarEssence is a front-end project I built to practice designing a product-focused landing page.

I wanted something minimal, elegant, and easy to navigate—where the focus stays on the earrings instead of too many UI elements.

---

## What I Focused On
- Keeping the layout clean and not cluttered  
- Making the design feel a bit “luxury” using beige tones  
- Ensuring it works well on different screen sizes  
- Displaying images properly using a 4:3 ratio  

---

## Built With
- HTML  
- CSS  
- JavaScript  

---

## What I Learned
While working on this, I improved on:
- Structuring a landing page  
- Using CSS variables for consistent styling  
- Handling images so they look good in a grid  
- Paying attention to spacing and alignment  

---

## Things I’d Improve
If I revisit this project, I’d:
- Add a shopping/cart feature  
- Improve animations and transitions  
- Add filtering for products  
- Maybe connect it to a backend
